Before executing any of the follow scripts, please read through this to know what each do

File 0-preprocessor runs a .c file through preprocessing and saves results into $CFILE
File 1-compiler, this script compiles a C file's content but doesn't link it
File 2-assembler is used to output the assembly code of C file main.c into a new file main.s
File 3-name is used to create an executable c file named 'cisfun'
File 4-puts.c is a script to print "Programming is like a multilingual puzzle" using the 'put' command
File 5-printf.c is a script using printf to display 'with proper grammer, but the outcome is a piece of art,'
File 6-size.c is a C script to print various sizes of different types of computer its compiled on
